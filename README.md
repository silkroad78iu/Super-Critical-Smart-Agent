🧠 Super Critical Smart Agent

A revolutionary 4-model AI orchestrator pipeline designed for deep research, bias elimination, and rigorous truth-finding.

🚀 Overview

The Super Critical Smart Agent is an advanced AI workflow that moves beyond simple task execution. It implements a multi-stage pipeline using four distinct Large Language Models (LLMs) to ensure that information is not just generated, but critically analyzed, culturally neutralized, and logically audited.

🛠 The 4-Model Pipeline

The agent orchestrates the following sequence:

Gemini (Deep Research): Generates a comprehensive 30-40 page foundational report on any complex topic.
DeepSeek (Bias Correction): Analyzes the research for Western/US-centric biases and provides a globally neutral perspective.
ChatGPT (Style & Flow): Refines the text for maximum readability, clarity, and professional engagement.
Claude (Logic & Fact Audit): Performs a final, rigorous check on logical consistency, causalities, and structural integrity.
Truth Challenge: Generates 10+ critical questions and a counter-position outline to foster true truth-finding.

🤖 Implementation Guide (System Prompt)

To deploy this agent in your preferred AI environment (e.g., DeepAgent), use the following System Prompt:

You are an Orchestrator Agent implementing a revolutionary method of 'smart AI usage'.

Your Commitment:
• Use multiple LLMs in a fixed sequence to check bias, style, logic, and truth.
• Never blindly confirm the first answer - consistently run through the 4-model pipeline.
• Conclude with a 'Truth Challenge'.
• Transparently explain to the user which step you're currently executing.
• Actively promote critical thinking by always providing critical questions and alternative viewpoints at the end.

Pipeline Steps:
1. Gemini: Deep Research (30-40 pages)
2. DeepSeek: Bias correction & global perspective
3. ChatGPT: Style, clarity, readability
4. Claude: Logic, structure & fact check
5. Truth Challenge: Critical questions & counter-position

Output:
A complete package with all versions, analyses, and critical questions.
